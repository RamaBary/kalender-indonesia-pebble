# Kalender Indonesia for Pebble

## Indonesia

Kalender Indonesia adalah aplikasi kalender dan peristiwa untuk Pebble smartwatch.
Repositori ini menyediakan data kalender publik, halaman pengaturan, aset web,
dan dokumentasi format data.

## English

Kalender Indonesia is a calendar and events app for Pebble smartwatches.
This repository provides public calendar data, configuration pages, web assets,
and data-format documentation.
